# 📝 Intro Component With Sign Up Form

Proyecto de maquetación web basado en el desafío **"Intro Component With Sign Up Form"** de Frontend Mentor.  
El objetivo fue construir una landing page responsive con un formulario de registro funcional, aplicando HTML semántico, SCSS y JavaScript moderno para validación de formularios.

---

## 📌 Descripción

Este proyecto consiste en una página promocional con un formulario de registro diseñado para captar nuevos usuarios mediante una interfaz visual clara y responsive.

Incluye validación personalizada de campos con retroalimentación visual en tiempo real, mostrando mensajes e iconos de error cuando los datos ingresados no son válidos.

Se puso especial énfasis en:

- Estructura semántica  
- Diseño responsive  
- Validación de formularios con JavaScript  
- Accesibilidad web  
- Código limpio y mantenible  
- Organización modular de estilos con SCSS  
- Experiencia de usuario (UX) mediante validación en tiempo real

## 🎯 Características principales

- 📱 Diseño completamente responsive para mobile y desktop  
- 📝 Formulario de registro con validación personalizada  
- ⚠️ Manejo dinámico de errores mediante JavaScript  
- 🖱️ Interactividad basada en eventos (`submit` e `input`)  
- 🔄 Validación en tiempo real mientras el usuario escribe  
- 🎨 Transiciones CSS para mejorar la experiencia visual  
- 🧩 Organización modular de estilos con **SCSS**  
- 🏷️ Uso de metodología **BEM** para una estructura CSS escalable  
- 📐 Layout responsive implementado con **CSS Grid**  
- 🔤 Tipografía fluida mediante `clamp()`  
- ♿ Implementación de accesibilidad con `label`, `autocomplete` y clase `sr-only`  
- ⚡ Entorno de desarrollo moderno con **Vite**  

---

## 🧠 Aprendizaje

Durante este proyecto puse en práctica conceptos clave como:

- Manipulación del DOM con JavaScript  
- Uso de `addEventListener()` para manejar eventos del formulario  
- Validación nativa de formularios mediante `validity.valid`  
- Implementación de validación personalizada con `novalidate`  
- Validación dinámica usando eventos `input` y `submit`  
- Manipulación de propiedades del DOM como `hidden`  
- Separación de responsabilidades mediante funciones reutilizables  
- Organización de estilos con arquitectura modular en SCSS  
- Diseño responsive con media queries  
- Uso de `CSS Grid` para layouts adaptables  
- Implementación de tipografía fluida mediante `clamp()`  
- Aplicación de metodología **BEM** para mantener escalabilidad del CSS
